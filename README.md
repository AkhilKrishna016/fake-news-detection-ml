🚀 Implemented Models

As part of this project, multiple supervised machine learning algorithms were systematically implemented and evaluated to analyze their effectiveness in fake news detection.

Day	Model	Description
Day 1	Logistic Regression	Baseline linear model for binary classification
Day 2	Naive Bayes	Probabilistic classifier optimized for text data
Day 3	Support Vector Machine	High-performance model for high-dimensional data
Day 4	Decision Tree	Rule-based model for interpretability
Day 5	Random Forest	Ensemble method to improve accuracy and reduce overfitting
Day 6	K-Nearest Neighbors	Instance-based learning using similarity measures

Each model was trained, tested, and compared using standard evaluation metrics to determine its suitability for text classification tasks.
